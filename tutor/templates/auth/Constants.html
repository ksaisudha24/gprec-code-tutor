Constants


Constants are often called literals because their value never changes. Constants are of two types:

1️⃣ Primary constants


➡ Integer Constant -> 426, -8000 etc.

➡ Real Constant -> 325.34, -48.652, 3.2e-5, -0.2e+3 etc.

➡ Character Constant -> 'A', 'a', '=', '1'


2️⃣ Secondary Constants


➡ Arrays

➡ Pointers

➡ Structures

➡ Unions


Rules for Constructing Integer Constants


✔ An integer constant must have at least one digit.

✔ It must not have not have a decimal point.

✔ It can be either positive or negative.

✔ If no sign precedes an integer constant it is assumed to be positive.

✔ No commas or blanks are allowed within an integer constant.

✔ The allowable range for integer constants is -32768 to 32767.

Ex: 1, 600


Rules for Constructing Real Constants


Real constants are often called Floating Point constants. The real constants could be written in two forms—Fractional form and Exponential form.

✔ A real constant must have at least one digit.

✔ It must have a decimal point.

✔ It could be either positive or negative.

✔ Default sign is positive.

✔ No commas or blanks are allowed within a real constant.

Ex: +325.34


Rules for Constructing Character Constants


✔ A character constant is a single alphabet, a single digit or a single special symbol enclosed within single inverted commas. Both the inverted commas should point to the left.

✔ For example, ’A’ is a valid character constant whereas ‘A’ is not.

✔ The maximum length of a character constant can be 1 character.

Ex.: 'A'
