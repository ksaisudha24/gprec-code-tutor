  Loop Control Structure

The basic attribute of an iteration or loop control structure is to be able to repeat some lines of code. The visual display of iteration creates a circular loop pattern when flowcharted, thus the word “loop” is associated with iteration control structures. Iteration can be accomplished with test before loops, test after loops, and counting loops. A question using Boolean concepts usually controls how often the loop will execute.

In 'C', iterative control structure is implemented using:

1. while loop

2. do-while loop

3. for loop

While Loop

while loop is used to repeat certain statements as long as the condition is true, when the condition is false, the while loop is quieted . This loop control structure is called as an entry controlled loop because only when the condition is true, all the statements are executed.

Syntax of while loop :

while (condition test) {

    //Statements to be executed repeatedly

    // Increment (++) or Decrement (--) Operation

}

next statements;

Write a program to check if a number is palindrome.

  A palindrome number is a number that reads same from both directions. e.g- 121, 1331, 41514, etc.

     #include   ‹stdio.h›

     int main () {

         int  n, r, t, rev  =  0;

         printf ( "Enter value of n" );

         scanf ( "%d" , &a );

         r  =  n ;

         while  ( n  !=  0 ) {

             t  =  n  %  10 ;

             rev  =  ( rev  *  10 )  +  r ;

             n  =  n  /  10 ;

         }

         if  ( t  ==  rev ) {

             printf ( "Palindrome" );

         }

         else  {

             printf ( "Not Palindrome" );

         }

         return 0 ;

     }


do-while Loop

The do-while loop is similar to while loop. In a do-while loop, the condition is tested at the end of the loop. So, even when the condition is false the body of the loop is executed at least once this is an exit controlled loop.

Syntax:

do {

    Set of statements;

}

while (condition);

next statement;

Write a program to generate the Fibonacci Sequence.

  The numbers in the fibonacci sequence start with 0 and 1 and the next number in the sequence is then found by adding the previous two numbers. e.g - 0, 1, 1, 2, 3, 5, 8, 13, 21.... and so on.

     #include   ‹stdio.h›

     int main () {

         int  n, tn, fn  =  0 , sn  =  1 , i  =  3 ;

         printf ( "Enter the value of n" );

         scanf ( "%d" , &a );

         printf ( "%d " , fn );

         printf ( "%d " , sn );

         do   {

             tn  =  fn  +  sn ;

             printf ( "%d " , tn );

             fn  =  sn ;

             sn  =  tn ;

             i  +=  1 ;

         }

         while  ( i  ‹=  n );

         return 0 ;

     }


Difference between while & do while :

A condition is tested before entering the loop. If the condition is false, the loop statements are not executed at all . It is an entry controlled loop . The condition is tested at the end of the loop, the statements are executed at least one even when the condition is false. It is an exit controlled loop.

for Loops

A for loop is a repetition control structure that allows you to efficiently write a loop that needs to execute a specific number of times.

Syntax:

for (initialisation; condition; increment / decrement) {

    statement(s);

}

Here is the flow of control in a 'for' loop −

1️⃣ The init step is executed first, and only once. This step allows you to declare and initialize any loop control variables. You are not required to put a statement here, as long as a semicolon appears.

2️⃣ Next, the condition is evaluated. If it is true, the body of the loop is executed. If it is false, the body of the loop does not execute and the flow of control jumps to the next statement just after the 'for' loop.

3️⃣ After the body of the 'for' loop executes, the flow of control jumps back up to the increment statement. This statement allows you to update any loop control variables. This statement can be left blank, as long as a semicolon appears after the condition.

4️⃣ The condition is now evaluated again. If it is true, the loop executes and the process repeats itself (body of loop, then increment step, and then again condition). After the condition becomes false, the 'for' loop terminates.

     #include   ‹stdio.h›

     int main () {

         int  i  =  1 , sum  =  0 ;

         for  ( int  i  =  1 ;  i  ‹=  10 ;  i ++) {
