Algorithm


What is Algorithm?


Before writing a program, a basic idea of how to solve the given problem is extremely necessary. An algorithm helps us have this basic idea. An algorithm is a set of instructions for solving a particular problem.

Characteristics of algorithms


 1️⃣  Algorithms should be clear and unambiguous.

 2️⃣  Algorithms must terminate after a finite number of steps.

 3️⃣  Algorithm’s effectiveness is measured in terms of time and space.


Types of algorithms


✔️ Sequential algorithm :

In a sequential algorithm, different steps occur in a sequence.

Eg: Finding average of the given numbers

 › Reading the numbers

 › Calculating the sum of the numbers

 › Calculating the average of the numbers


✔️ Conditional algorithm :

A conditional pattern is the one where different steps are executed based on a condition.

Eg: To find if a number is odd or even

Read number

if number % 2 == 0 -> it is an even number

else -> it is an odd number

( % is used to find the remainder of two numbers)


✔️ Iterative algorithm :

In this algorithm, a task is performed more than once(repeated).

Let us look at an example of how to write an algorithm to make you understand better.

Printing numbers from 1 to 10.

Start

Declare a variable i;

Assign i = 1;

if(i ‹= 10)

Yes-> print i;

i = i + 1; goto step 4

No-> goto step 5

Stop
