Keywords


Keywords are the words whose meaning has already been explained to the C compiler. The keywords cannot be used as variable names because if we do so we are trying to assign a new meaning to the keyword, which is not allowed by the computer.

There are only 32 keywords available in C which are :

  1. ➡auto
  2. ➡break
  3. ➡case
  4. ➡char
  5. ➡const
  6. ➡continue
  7. ➡default
  8. ➡do
  9. ➡double
  10. ➡else
  11. ➡enum
  12. ➡extern
  13. ➡float
  14. ➡for
  15. ➡goto
  16. ➡int
  17. ➡long
  18. ➡register
  19. ➡return
  20. ➡short
  21. ➡signed
  22. ➡sizeof
  23. ➡static
  24. ➡struct
  25. ➡switch
  26. ➡typedef
  27. ➡union
  28. ➡unsigned
  29. ➡void
  30. ➡volatile
  31. ➡while
