  Decision Control Structure

The decision control structure controls the flow of the program to specific parts depending on the logical descion.

It mainly has -

  • if statement
  • if - else statement
  • else if

If statement

An ‘if’ statement is a selective control structure that executes a set of statements based on a condition.

Simple ‘if’ statement :- In a simple ‘if’ statement, a condition is tested,

i) If the condition is true, a set of statements are executed.

ii) If the condition is false,the statements are not executed and the program control goes to the next statement that immediately follows the ‘if’ block.

Syntax:-

if(condition) {

Set of statements

}

Next statement;

 Note:- The ‘if’ statement does not end with a ; where there is only one statement followed by ‘if’ there is no need for braces.

But it is a good programming practice to use curly braces in such cases also. Always use ‘if’ statement with curly braces.

If-else statement

In simple if statement, when the condition is true, a set of statements are executed but when it is false there is no alternative set of statements.The statement ‘else’ provides the same.

Syntax :-

if (condition) {

    Statement-set 1 ;

}

else {

    Statement -set 2 ;

}

Next Statement ;

Else-if Statement :- (else if ladder)

else-if statement is to check for a sequence of conditions when one condition is false it checks for the next condition and so on. When all the conditions are false, the else block is executed.

Each condition is tested in sequence until one evaluates to true.

The statements in the conditional block are executed and other if statements are skipped.

Syntax :-

if(condition-1) {

    statement set-1;

}

else if(condition-2) {

    statement set-2;

}
